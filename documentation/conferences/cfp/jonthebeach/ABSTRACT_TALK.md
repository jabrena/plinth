Technical Workshop on AI Tooling for Java Development

Description
For nearly three decades, enterprise Java software has been implemented manually by software engineers.
In 2023, a new generation of AI-powered tools emerged, and tools like Cursor AI, Claude Code, and others are now changing the way software is developed.

This workshop is designed for all levels and aims to cover in a practical way the following topics:

- Introduction
- How do you develop?
- How do you use an agent-driven engineering workflow?
- Evolving the SDLC
- Concepts
- Demos

In the demos, participants will understand and learn about the power of AI agent harnesses in daily work across classic areas such as analysis, design, and implementation.

The demos will explain how to create a good `AGENTS.md`, how to build a solid collection of MCP servers, and how to use sub-agents in implementations and skills.

The talk explains the main concepts using examples based on enterprise frameworks such as Spring Boot, Quarkus, and Micronaut, and the workshop will be delivered in English.
