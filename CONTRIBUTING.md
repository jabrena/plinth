# Contributing

## Generator

The entire set of cursor rules is autogenerated by XML files in the project `system-prompts-generator`.
[All XML files](./system-prompts-generator/src/main/resources/system-prompts/) use the PML Schema ([pml.xsd](https://jabrena.github.io/pml/schemas/0.5.0/pml.xsd)) and those XML documents are transformed into Markdown with frontmatter using a [Java class](./system-prompts-generator/src/main/java/info/jab/pml/CursorRulesGenerator.java) and an [XSL file](./system-prompts-generator/src/main/resources/system-prompts.xsl) specialized in the Cursor rule format.

If you have the idea to contribute, review the whole process in detail:

```bash
./mvnw clean verify -pl system-prompts-generator   # Pass tests
./mvnw clean install -pl system-prompts-generator  # Pass tests & copy new .md files into .cursor/rules
```

When you feel confident with the process, fork the repository and try to create new XML documents. Models will help you because an XML file is more rigid than natural language and it has `a common vocabulary` to create prompts.

## Skills generator

Agent skills are generated from the cursor rules in the project `skills-generator`. To contribute skills:

```bash
./mvnw clean verify -pl skills-generator   # Pass tests
./mvnw clean install -pl skills-generator # Generate skills into skills/
npx skill-check skills                     # Validate generated skills
```

When you feel confident with the solution, send a PR.
